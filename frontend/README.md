# Frontend - InoFarm

Esta pasta contém todo o código frontend do projeto InoFarm.

## Estrutura

- **HTML**: Páginas principais da aplicação
- **CSS**: Estilos das páginas
- **img/**: Imagens e assets do projeto
- **HeaderComum/**: Componente de header para páginas comuns
- **HeaderGerente/**: Componente de header para páginas do gerente

## Páginas Principais

- `index.html` - Página principal de listagem de frutas
- `login1.html` - Página de login
- `cadastroTeste_Pessoas.html` - Cadastro de pessoas
- `cadastroTeste_Acesso.html` - Cadastro de acesso
- `quem-somos.html` - Página sobre a empresa
- `CarrinhoFinaliza.html` - Finalização do carrinho
- `senha.html` - Página de senha para clientes
- `senhaGerente.html` - Página de senha para gerente
- `vendas.html` - Dashboard de vendas (gerente)
- `gerente_produtos.html` - Gerenciamento de produtos (gerente)
- `entrada_saida.html` - Registros de entrada e saída (gerente)

## Como Executar

Abra o arquivo `index.html` em um navegador ou use um servidor HTTP local:

```bash
# Python 3
python -m http.server 8000

# Node.js (com http-server)
npx http-server
```

Acesse `http://localhost:8000` no navegador.
